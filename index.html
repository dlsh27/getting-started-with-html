The Worldwide Web is primarily a collection of HTML documents.

Major Pieces of the WWW

 1) Your computer and browser (MS IE, Chrome, Firefox, etc.).
2) The Internet (world's largest computer network).
3) Server (computer that receives request using HTTP protocol and sends information back to browser).

HTML documents are the heart of the Web.
HTML documemts are made up of elements. Browsers recognize these elements through the use of tags.

Tags can be used to format text, add images and or create links to web pages.
Here are examples: Bold, Italics or Sports.

There are two types of elements - Inline & Block. Block elements create an invisible box around content.
